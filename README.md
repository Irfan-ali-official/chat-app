
---

## 🔄 How It Works

1. Frontend connects to WebSocket server
2. Users send messages from React UI
3. Backend receives messages via WebSocket
4. Backend broadcasts messages to all connected users instantly
5. Messages appear in real-time without refreshing

---

## 🌐 Real-Time Communication

This application uses **WebSocket protocol**, which enables:

- Instant message delivery
- Persistent connection between client and server
- Real-time updates without polling

 ## Setup and running this app
 * cd backend
 * npm install
 * npm start or nodemon server.js
 * Backend server can be seen on port localhost:8081
 * cd frontend
 * npm install
 * npm start
 * Frontend Could be seen on port http://localhost:5173/
